# metadev

Runs on io.js.

NVM makes io.js easier - [Node Version Manager](https://github.com/creationix/nvm) - ctrl+f iojs

```bash
# SETUP
$ git clone https://github.com/metagen/metadev && cd metadev
$ npm install

# SERVE
./bin/serve.js

# DEVELOP
npm install -g nodemon
nodemon ./bin/serve.js
```
